NOTRE SOLUTION

Un pipeline 100% automatique

  • check_circle   Lit la BAL bounces en temps réel (Postfix) ou en cron IMAP
  • check_circle   Classifie avec un moteur hybride règles + Claude Haiku
  • check_circle   Historise les soft bounces cross-jours (anti-spam)
  • check_circle   Génère 3 CSV : to_delete, to_pause, to_modify
  • check_circle   Envoie le rapport quotidien + push direct au CMS
  • check_circle   Alerte sur Slack en cas d'anomalie

webhook

WOW FEATURE · 1 / 3

smart_toy  Self-improving rules

Le LLM apprend au système à se passer de lui.

Mécanique

  • 1. Bounce ambigu → le LLM est appelé
  • 2. Il classe ET propose une regex générique
  • 3. La regex va dans rule_suggestions
  • 4. Tu cliques Adopter sur le dashboard
  • 5. Bounces similaires → matchés par règles → 0 appel LLM

Pourquoi c'est unique

  • trending_down  Coût LLM diminue avec le temps
  • speed  Pipeline plus rapide (règles = µs, LLM = secondes)
  • verified  Maintenance facile (rules versionnées en JSON)
  • handshake  L'humain garde le contrôle (validation avant adoption)
